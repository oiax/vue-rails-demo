vue-rails-demo
==============

Vue.js と Rails 5.1 を組み合わせた Web アプリケーションのデモ

## インストール方法

```bash
git clone https://github.com/oiax/vue-rails-demo.git
cd vue-rails-demo
bundle
yarn
bin/rake db:setup
```
